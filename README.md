# Projet
Ceci est juste une base de projet C++ utilisant la SFML avec un moyen de le compiler sous Windows ou Linux

# Compilation
## Linux :
make

## Windows :
mingw32-make win

## .exe a partir d'un linux :
make linux-win 

# Lancement
## Linux :
make run

## Windows :
mingw32-make winrun
OU
Double clique sur le .exe dans windows_executable
